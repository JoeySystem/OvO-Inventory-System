# OvO Inventory System 仓库员速查版

> 适合仓库员、车间领料人员快速上手  
> 目标：只看最常用操作，不讲复杂系统设计

## 1. 先按问题找入口

### 采购到货了，怎么入库

- 路径：`入库 > 收货入库`
- 操作：选仓库 -> 选物料 -> 填数量 -> 填单价或总价 -> 选供应商 -> 点 `一键提交并执行记账`

### 想查刚录的入库单

- 路径：`入库 > 采购收货单`
- 操作：按单号 / 物料 / 日期筛选 -> 点单据号 -> 看详情

### 单据录错了，怎么处理

- `草稿`
  - 点 `编辑草稿`
- `已提交`
  - 点 `编辑已提交`
  - 填修改原因
  - 保存后会退回草稿，再重新提交
- `已执行`
  - 先点 `撤销执行`
  - 再点 `编辑已提交`
- `已记账`
  - 先点 `反记账`
  - 再点 `撤销执行`
  - 再点 `编辑已提交`
- 整张单完全错了
  - 点 `红冲`

### 只发一种料，怎么出库

- 路径：`出库 > 发料出库`
- 操作：选仓库 -> 选一个物料 -> 填数量 -> 点 `一键提交并执行记账`

### 按整套 BOM 发料，怎么做

- 路径：`出库 > 按BOM发料`
- 操作：打开目标 BOM -> 在 `缺料检查` 里选仓库和生产数量 -> 点 `按BOM发料`

### 想知道一张历史采购单买了什么

- 路径：`入库 > 历史采购记录`
- 操作：点单据号 -> 看 `这张单买了哪些物料`

### 想查某个物料还有多少库存、多少钱

- 路径：`物料`
- 操作：搜索物料 -> 列表看价格 -> 点详情看库存、最近采购价、供应商报价

### 想确认单据有没有真正生效

- 最稳的判断：
  - 看状态是不是 `posted`
  - 再看动作流水里能不能查到

## 2. 先记住这几个入口

左侧导航常用的是：

- `入库`：收货入库、完工入库、退料回库
- `出库`：发料出库、销售发货、按BOM发料
- `调拨`：仓间调拨
- `盘点`：盘点调整
- `物料`：查物料库存、查详情
- `报表 > 动作流水`：查历史动作

## 3. 收货入库怎么做

路径：

- `入库 > 收货入库`

步骤：
1. 选仓库
2. 搜索并选择物料
3. 填数量
4. 填往来方或备注
5. 日常使用直接点 `一键提交并执行记账`

适用场景：
- 采购到货
- 完工入库
- 退料回库

如果要专门查收货类单据：
- `入库 > 采购收货单`

## 4. 发料出库怎么做

路径：

- `出库 > 发料出库`

步骤：
1. 选仓库
2. 搜索并选择物料
3. 填数量
4. 填用途或备注
5. 直接点 `一键提交并执行记账`

注意：
- 出库数量不能超过当前可用库存

如果要按生产配方一次性领整套料：
- `出库 > 按BOM发料`
- 进入后打开对应 BOM
- 在 `缺料检查` 里选仓库和生产数量
- 再点 `按BOM发料`

## 5. 仓间调拨怎么做

路径：

- `调拨 > 仓间调拨`

步骤：
1. 选源仓
2. 选目标仓
3. 选物料
4. 填调拨数量
5. 填参考号或备注
6. 点 `一键提交并执行记账`

适用场景：
- 主仓调到车间仓
- 车间退回主仓

## 6. 盘点调整怎么做

路径：

- `盘 > 盘点调整`

步骤：
1. 选仓库
2. 选物料
3. 看账面库存
4. 输入实盘数量
5. 检查调整差值
6. 点 `一键提交并执行记账`

注意：
- 盘点调整会直接改库存
- 备注建议写清原因，比如“月度盘点”

如果要专门查调整类单据：
- `盘点 > 库存调整单`

## 7. 单据怎么查

在 `入库 / 出库 / 调拨 / 盘点` 页面里都能查。

另外，以下单据已有独立查询页：
- `入库 > 采购收货单`
- `盘点 > 库存调整单`
- `生产 > 生产领料单`
- `生产 > 完工入库单`
- `生产 > 退料回库单`
- `生产 > 生产异常单`

常用方法：
1. 进入对应模块
2. 切到 `单据视图`
3. 用单号 / 物料 / 日期筛选
4. 看状态是不是 `posted`

状态简单理解：
- `draft`：草稿
- `submitted`：已提交，未真正记账
- `executed`：已执行
- `posted`：已记账，正式生效
- `voided`：已作废

日常判断标准：
- 想确认这张单是不是已经生效，主要看是不是 `posted`

## 8. 物料怎么查

路径：

- `物料`

可搜索：
- 名称
- 编码
- 拼音
- 规格

打开方式：
- 直接点物料整行
- 或点 `详情`

在物料详情里可以看：
- 当前库存
- 最近流水
- 被哪些 BOM 用到
- 供应商报价

如果物料供应风险高，还可以点去：
- `报表 > 业务报表 > 高风险供应治理`

## 9. 销售发货怎么做

路径：

- `出库 > 销售发货`

步骤：
1. 点 `新建销售发货`
2. 填客户信息
3. 选物料
4. 填数量
5. 保存并按状态流转

说明：
- 新建时默认会有一行空明细
- 发货状态到 `已发出` 后才真正扣库存

## 10. 生产异常怎么查

路径：

- `生产 > 生产异常单`

适合查：
- 报废
- 补料
- 超领
- 差异

如果需要跟进处理，可以在详情里看：
- 当前治理状态
- 责任人
- 处理备注
- 是否已闭环

## 11. 查历史动作怎么做

路径：

- `报表 > 动作流水`

适合查：
- 某个物料最近怎么变动
- 某张单据有没有真正影响库存

常用筛选：
- 日期
- 仓库
- 物料
- 单号

## 12. 常见问题

### 页面提示成功，为什么库存没变

通常原因：
- 只是提交了单据
- 没有执行记账

处理：
- 去查单据状态
- 是否已经到 `posted`

### 为什么查不到刚录的单

先检查：
- 是否在正确模块
- 当前是不是 `单据视图`
- 日期范围是不是太窄

### 为什么某个料不能出库

可能原因：
- 库存不足
- 仓库选错
- 选到的不是正确物料

### 为什么一搜索就找不到物料

建议优先试：
- 编码
- 名称关键词
- 拼音
- 规格

## 13. 日常最常用的 5 个动作

仓库员日常最常用的是：

1. 收货入库
2. 发料出库
3. 仓间调拨
4. 盘点调整
5. 查单据状态

先熟练这 5 项，就能完成大部分日常操作。

## 14. 仓库员最常问的 10 个问题

### 1. 采购到货了，第一步去哪

- 进 `入库 > 收货入库`

### 2. 单据上有多个物料，怎么录

- 在同一张入库单里逐条添加物料明细

### 3. 单价和总价怎么算

- 填 `数量 + 单价`，系统自动算总价
- 填 `数量 + 总价`，系统自动反算单价

### 4. 供应商没带出来怎么办

- 先确认这个物料有没有维护默认供应商
- 没有的话，可以先手工填
- 后续到 `物料详情` 里补完整供应商信息

### 5. 刚录完的单据去哪里查

- 入库单去 `入库 > 采购收货单`
- 出库单去对应模块的单据查询页
- 也可以去 `报表 > 动作流水` 查

### 6. 为什么提交成功了，库存没变

- 因为可能只是提交了，还没真正记账
- 最稳的判断是看状态是不是 `posted`

### 7. 单据录错了怎么办

- `草稿`：点 `编辑草稿`
- `已提交`：点 `编辑已提交`
- `已执行`：先点 `撤销执行`
- `已记账`：先点 `反记账`
- 整张单都错了：点 `红冲`

### 8. 想一次性按整套物料发料，去哪

- 进 `出库 > 按BOM发料`

### 9. 想知道某个料现在还有多少

- 进 `物料`
- 搜索这个物料
- 在列表或详情里看当前库存

### 10. 想知道一张历史采购单买了什么

- 进 `入库 > 历史采购记录`
- 点单据号
- 看 `这张单买了哪些物料`
